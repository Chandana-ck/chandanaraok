# 🚀 Projects

## 🔍 Fake News Detection System

A multimodal AI system designed to identify misinformation by analyzing news content across different input formats such as text, images, and speech.

The system evaluates credibility using contextual analysis and artificial intelligence techniques.

Key components include:

- Natural Language Processing for textual analysis  
- Optical Character Recognition for extracting text from images  
- Speech-to-text processing for spoken inputs  
- AI-based classification using large language models  
- Explanation-based outputs to improve transparency  

This work resulted in the research publication:

**Fake News Detection: An AI-Based Approach**  
Published in *Journal of Emerging Technologies and Innovative Research (JETIR)*.

## ⚙️ FOSYS — AI Workflow Automation Platform

🏆 **Best Paper Award — IC-AISMART 2025**

FOSYS is an AI-powered workspace designed to improve collaboration and workflow tracking in SCRUM-based software development teams.

The system connects meeting discussions, task management, and development activity within a unified platform.

Core capabilities include:

- AI extraction of tasks from SCRUM meeting transcripts  
- Structured task lifecycle tracking and management  
- Integration with GitHub Pull Request events using webhooks  
- Validation of task completion based on code merges  
- Role-based dashboards for developers and managers  

The platform improves workflow visibility, accountability, and coordination within collaborative development environments.
