# ✨ Selected Work

## 🔍 Fake News Detection System

An AI-based system designed to identify misinformation across digital platforms by analyzing news content from multiple input formats.

The system evaluates credibility by combining different AI techniques and data processing methods.

**Key capabilities**

🧠 Natural Language Processing for contextual text analysis  
🖼️ Optical Character Recognition to extract text from images  
🎙️ Speech-to-text conversion for spoken news inputs  
⚡ AI classification using Large Language Models  
📊 Explanation-based results to improve transparency

📄 **Publication**  
*Fake News Detection: An AI-Based Approach*  
Published in **Journal of Emerging Technologies and Innovative Research (JETIR)**

## ⚙️ FOSYS — AI Workflow Automation Platform

🏆 **Best Paper Award — IC-AISMART 2025**

FOSYS is an AI-powered workspace designed to improve coordination in **SCRUM-based software development teams**.

The system connects meeting discussions, task workflows, and development activity within a unified platform.

**Platform highlights**

🧩 AI extraction of tasks from SCRUM meeting transcripts  
📋 Structured task lifecycle tracking  
🔗 GitHub Pull Request integration through Webhooks  
✅ Automatic validation of tasks based on code merges  
👥 Role-based dashboards for development teams

The platform improves **workflow visibility, accountability, and collaboration** in development environments.
